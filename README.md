# CestUnBossCa ??
C'est un boss ça ?? est un petit jeu où tu dois deviner si le personnage sur l'image est un boss ou non.

## Stack technique
Next.js avec React, prêt pour un déploiement sur Vercel.

## Lancer le projet
1. Installer les dépendances :
   ```bash
   npm install
   ```
2. Démarrer le serveur de développement :
   ```bash
   npm run dev
   ```
3. Ouvrir `http://localhost:3000` dans ton navigateur.

## Fonctionnalités
- Demande le pseudo du joueur
- Montre 10 images successives
- Permet de répondre "Boss" ou "Pas boss"
- Affiche le score final et un récapitulatif des réponses

## Remarques
Les images utilisées sont des visuels de test. Tu peux remplacer `lib/images.js` par tes propres images locales ou externes.
